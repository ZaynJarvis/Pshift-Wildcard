# WildCard

## Pre-requisites

- Install `yarn`.

## How to Run

- Run `yarn install`.

- Run `yarn start`.

## How to Install Packages

```
$ yarn add your-package-name --save
```

## How to Upgrade Packages

```
$ yarn upgrade your-package-name
```

## Where to Add Custom Global CSS

```
Layout/style.css
```

> Avoid unless absolutely necessary
